# AI-Powered Code Quality & Intelligence Platform

A full-stack, containerized platform that combines rule-based static code analysis and AI-driven semantic code intelligence to enhance software quality and developer productivity.

# This system integrates:

🔍 Static Analysis via SonarQube
🤖 AI-based Semantic Code Review
⚙️ Spring Boot Backend APIs
🎨 Angular Frontend Dashboard
🐳 Docker-based Deployment

# 🧠 Key Features

# 🔍 Static Code Analysis

Detects bugs
Identifies vulnerabilities
Flags code smells
Calculates cyclomatic complexity
Estimates technical debt

# 🤖 Semantic AI Analysis

Explains Sonar issues in simple language
Suggests refactored code improvements
Analyzes logic structure
Estimates time complexity
Recommends design optimizations

# 📊 Dashboard

Visual breakdown of code issues
Side-by-side original vs improved code
Aggregated quality scoring
